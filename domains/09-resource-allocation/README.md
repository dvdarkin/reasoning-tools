# Resource Allocation

Reasoning tools for distributing scarce resources efficiently and equitably.

## Domains in This Category

1. **Economics** - Opportunity cost, comparative advantage, marginal analysis, market mechanisms
2. **Operations Research** - Linear programming, optimization, queueing theory, scheduling
3. **Portfolio Management** - Diversification, risk-return tradeoffs, rebalancing, asset allocation
4. **Distributive Justice** - Fairness principles, equality vs equity, priority setting

## Common Themes

- Recognizing and accounting for opportunity costs
- Optimizing under constraints
- Balancing efficiency with equity
- Managing tradeoffs and diminishing returns
- Allocating to highest-value uses

## Cross-Domain Tools

Tools that appear across multiple domains in this category:
- Opportunity cost analysis
- Marginal analysis and diminishing returns
- Constraint identification and relaxation
- Diversification and hedging
- Pareto efficiency and improvement
