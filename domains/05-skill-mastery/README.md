# Skill Acquisition & Mastery

Reasoning tools for learning, practice, and expertise development.

## Domains in This Category

1. **Learning Theory (Cognitive Science)** - Spacing effects, retrieval practice, interleaving, cognitive load
2. **Sports Science** - Deliberate practice, motor learning, periodization, performance feedback
3. **Expertise Studies** - Chunking, pattern recognition, mental models, 10,000 hour rule

## Common Themes

- Optimizing practice for retention and transfer
- Building mental models and pattern libraries
- Managing cognitive load and attention
- Using feedback effectively
- Progressing from novice to expert performance

## Cross-Domain Tools

Tools that appear across multiple domains in this category:
- Deliberate practice
- Spaced repetition and retrieval
- Interleaving and variation
- Immediate feedback loops
- Chunking and pattern recognition
