# Creative Generation

Reasoning tools for generating novel solutions and innovations.

## Domains in This Category

1. **Design Thinking** - Human-centered design, rapid prototyping, iterative refinement
2. **Evolutionary Biology** - Variation, selection, adaptation, exaptation

## Common Themes

- Generating diverse alternatives
- Combining existing elements in novel ways
- Iterative exploration and refinement
- Balancing exploration with exploitation
- Learning from failure and variation

## Cross-Domain Tools

Tools that appear across multiple domains in this category:
- Divergent then convergent thinking
- Rapid prototyping and iteration
- Recombination and variation
- Selection pressure and fitness landscapes
- Constraint as creative catalyst
