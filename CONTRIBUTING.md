# Contributing

Want to add a domain? We welcome contributions.

## Adding a New Domain

1. **Follow the template**: [meta/extraction-template.md](meta/extraction-template.md)
2. **Meet the quality bar**: [meta/quality-criteria.md](meta/quality-criteria.md)
3. **Open a pull request**

## Other Contributions

- Typo fixes and clarifications welcome
- Improvements to existing tools (better examples, additional failure modes, stronger citations)
- Suggestions for Phase 2 domains via issues

## Style

- Match the existing format and tone
- Cite serious academic literature, not pop science
- Focus on transferable mental operations, not domain-specific facts
